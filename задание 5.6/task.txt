

Зачастую возникает задача обезопасить данные сайта от копирования. Её сложно выполнить на 100%, однако простые способы доработки верстки позволяют добиться уже каких-то результатов. Один из методов для усложнения копирования картинок – это натягивание другого элемента поверх текущего. Таким образом, когда пользователь будет кликать правой кнопкой мыши по "картинке", на самом деле клик будет происходить по натянутому поверх элементу.
Давайте попробуем это реализовать.

    Добавьте в тег img атрибут src со значением: /static/sber/task_19/image.jpeg
    Добавьте в тег с классом image тег div с классом protect.
    Тегу с классом image добавьте относительное позиционирование.
    Тегу с классом protect добавьте абсолютное позиционирование и растяните его на всю ширину и высоту родительского элемента.
    Переместите элемент с классом protect в верхний левый угол родительского элемента.
