

    В тег select добавьте атрибут name со значением type.
    В тег select добавьте тег option, внутрь которого напишите монокристаллический. У тега option должен быть атрибут value со значением mono.
    В тег select добавьте тег option, внутрь которого напишите поликристаллический. У тега option должен быть атрибут value со значением poli.
    В тег select добавьте тег option, а внутрь тега напишите кремниевые. У тега option должен быть атрибут value со значением silicon.
    Для тега option со значением value="mono" укажите атрибут selected.
