 Укажите внешнему div элементу класс facts.
Для элемента с классом facts укажите display: grid;.
С помощью стиля grid-template-columns сформируйте 3 столбца, где столбец посередине - в 2 раза шире боковых. Используйте единицы измерения fr.

Внимание! Из-за особенностей тестирующей системы для решения задачи необходимо использовать только положительные значения в grid-column без использования span
